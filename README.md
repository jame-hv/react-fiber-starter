## Run Locally

Clone the project

```bash
  git clone git@github.com:HoangVuAB/react-three-js-starter-kit.git
```

Go to the project directory

```bash
  cd react-three-js-starter-kit
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
